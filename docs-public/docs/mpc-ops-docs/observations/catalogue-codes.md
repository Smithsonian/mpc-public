# Catalogue Codes

Column 72 on the standard MPC1992 observation record encodes the astrometric reference catalogue as a single character. This column was originally added by Gareth Williams for doctoral research. Starting in mid-2001, automated processing routines populated this field based on observer-reported catalogue information in submission headers. Williams also manually added catalogue codes to earlier observations.

New submissions should leave column 72 blank, as the catalogue information is added during processing.

The proper reference is: G. V. Williams, 2012, "Minor Planet Astrophotometry", PhD thesis, Open University. [2012PhDT.........7W]
