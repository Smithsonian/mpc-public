# MPC Submission Information

For those wishing to submit astrometric observations for publication in the Minor Planet Circulars, information is available below on the format of astrometric observations and how to submit them.


## Submitting Observations

Observations should be formatted per the guidelines below and sent to [`obs@cfa.harvard.edu`](mailto:obs@cfa.harvard.edu).

- [Command-line submissions via curl](../observations/command-line-submissions.md) are also supported.
- New comet reports should be directed to the MPC (per the 2015 IAU General Assembly task redistribution).
- Visual comet discovery reports can be submitted via the [web form](https://minorplanetcenter.net/mpcops/submissions/cometary/).
