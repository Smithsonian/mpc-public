## Naming

Asteroid naming is handled by the IAU Working Group on Small Body Nomenclature (WGSBN), not by the MPC.

- [WGSBN website](https://www.wgsbn-iau.org/)
- [WGSBN naming rules and guidelines (PDF)](https://www.wgsbn-iau.org/documentation/NamesAndCitations.pdf)
- [WGSBN Bulletin](https://www.wgsbn-iau.org/files/Bulletins/index.html)
- [IAU Planetary System Nomenclature](https://www.iau.org/public/themes/naming/#satellites) (natural satellite naming)
