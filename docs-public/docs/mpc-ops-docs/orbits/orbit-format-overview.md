# Export Format for Orbits of Comets and Minor Planets

This page describes the format used for the export version of orbits of comets and minor
planets. These are the formats used in the Extended Computer Service and in the Minor
Planet Ephemeris Service.

Orbit computers wishing to submit orbits for publication in the *Minor Planet Circulars*
or *International Astronomical Union Circulars* are advised that the submission format is
different to that described below. It should also be noted that routine improvements of
known objects cannot be accepted from outside contributors.
