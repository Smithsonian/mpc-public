# Publishable Notes for Orbits of Minor Planets

This page documents the alphabetic codes used in orbit notes and residual blocks for
minor planet orbits.

---

## Orbit Note Codes

| Code | Meaning |
|------|---------|
| D    | Double (or multiple) designation(s) |
| E    | Eccentricity assumed |
| F    | = D + E (double designation with assumed eccentricity) |

---

## Residual Block Codes

| Code  | Meaning |
|-------|---------|
| ( )   | Brackets surround the residuals of observations rejected from the solution |
| Y     | Observation published to precision of 0s.1 in R.A. and 1" in Decl. |
| X     | Observation published to precision of < 1s in R.A. or < 0'.1 in Decl. |
